# Employee Attendance Management System

A modern **Employee Attendance Management System** designed to simplify workforce attendance tracking using **GPS location verification**, **webcam image capture**, and **role-based access control**. The application enables organizations to monitor employee attendance, manage daily tasks, and track work history through a secure and responsive web interface.

---

## Overview

The Employee Attendance Management System is a full-stack web application that digitizes the attendance process by replacing manual registers with location-aware and image-based verification. Employees can securely mark their attendance, complete assigned tasks, and view attendance history, while administrators can monitor employee activities, assign tasks, and generate reports.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/employee-attendance-management-system.git
```

Navigate to the project directory

```bash
cd employee-attendance-management-system
```

Install dependencies

```bash
npm install
```

Configure environment variables

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Run the development server

```bash
npm run dev
```
